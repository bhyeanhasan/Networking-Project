VoIP- IP Telephony System Network Design

##### CONFIG STEPS ######

1. Network Design.
2. Basic settings to all devices plus ssh on the routers.
3. VLANs assignment plus all access and trunk ports on the switchs.
4. Subnetting and IP addressing.
5. Static IP address to server room devices.
6. DHCP server device configurations.
7. Configure DHCP for Voice.
8. Inter-VLAN routing on the routers plus ip dhcp helper address.
9. OSPF on the routers.
10. Configure VoIP configuration in all routers.
11. Dial peering configuration in all routers.
12. Verifying and testing configuration.
